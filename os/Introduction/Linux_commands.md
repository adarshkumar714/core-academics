# Linux Commands

### Chmod command
```
  r w x r - x r - x
_ _ _ _ _ _ _ _ _ _
```

* 3 type of categories
    * U - user
    * G - group
    * O - others

* 3 types of permissions
    * r - read (4)
    * w - write (2)
    * x - execute (1)
    * 665 -> -rw-rw-r-x

* ***blocks***
  * 1st block to identify directory or file: d/-
  * next 3 blocks for USER
  * next 3 blocks for GROUP
  * next 3 blocks for OTHER
